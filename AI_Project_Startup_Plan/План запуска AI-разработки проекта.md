Данное руководство представляет собой модульный план для максимально качественного, организованного и грамотного старта проекта с использованием ИИ-агентов (в частности, Claude Code). План составлен на основе лучших практик, подходов Spec-Driven Development, Domain-Driven Design (DDD), методологии BMAD и современных экосистем (`everything-claude-code`, `claude-code-templates`).

В этом руководстве каждый шаг выделен в отдельный документ с подробным описанием, примерами использования и командами для запуска.

## Оглавление шагов

1. [Шаг 1: Установка и базовая настройка среды](steps/01_environment_setup.md) 🔴
   Отказ от классического чата в пользу работы в терминале напрямую с кодовой базой. Использование готовых шаблонов для быстрого и безопасного старта.
2. [Шаг 2: Создание базовых контрактов проекта](steps/02_basic_contracts.md) 🔴
   Единый источник правды: `CLAUDE.md`, `rules/`, `SKILL.md`, `AGENTS.md`, `DESIGN.md`.
3. [Шаг 3: Настройка системы управления контекстом и задачами](steps/03_state_management.md) 🔴
   Внедрение Memory Bank или Beads для предотвращения "снежного кома контекста" и потери фокуса.
4. [Шаг 4: Привлечение специализированных ролей](steps/04_specialized_roles.md) 🟡
   Установка `everything-claude-code` и использование ролей Архитектора, Планировщика, Разработчика, Ревьюера и Bug Hunter (на базе фреймворка Superpowers).
5. [Шаг 5: Spec-Driven Development (Планирование и Спецификации)](steps/05_spec_driven_development.md) 🔴
   Жесткая блокировка написания кода до завершения этапа проектирования (`/search-first`, `/plan`).
6. [Шаг 6: Настройка гейтов качества, Безопасности и TDD](steps/06_quality_gates_tdd.md) 🔴
   Внедрение фреймворка Superpowers, написание тестов до кода, автоматические хуки и AgentShield.
7. [Шаг 7: Кодирование субагентами (Swarms) и Мультисэмплинг](steps/07_swarms_multisampling.md) 🟡
   Делегирование задач узконаправленным субагентам и использование Agent Mail.
8. [Шаг 8: Многоуровневая верификация, Code Review и Continuous Learning](steps/08_verification_learning.md) 🔴
   Code Review, автоматическое извлечение "инстинктов" через Skill Creator и формирование новых навыков.
