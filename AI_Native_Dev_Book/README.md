# Учебник: AI-Native Dev

Добро пожаловать в учебник по AI-Native разработке, основанный на концепциях курса JavaRush University и дополненный материалами из нашей базы знаний.

Этот учебник предназначен для того, чтобы помочь вам перейти от традиционного программирования к разработке с использованием AI-агентов.

## Оглавление

1. [Глава 1: Среда, инструменты и основы мышления](Chapter_1_Environment_and_Mindset.md)
   * Настройка окружения (Claude Code)
   * Методология работы (как «думать задачами»)
   * Документация проекта (`CLAUDE.md`)
   * Task Specification (составление задач)
   * Context Management (управление контекстом)
   * Анализ кода
2. [Глава 2: Продвинутые инструменты и автоматизация](Chapter_2_Advanced_Tools_Automation.md)
   * Subagents (делегирование задач)
   * MCP (Model Context Protocol)
   * Hooks (автоматизация после коммитов)
   * Практические кейсы (тесты, рефакторинг, ревью, дебаг)
3. [Глава 3: Полный цикл разработки и запуск продукта](Chapter_3_Full_Cycle_and_Deployment.md)
   * Workflow «Issue-to-PR»
   * Рефакторинг
   * DevOps для разработчика (CI/CD, мониторинг)
   * Запуск продукта с нуля
4. [Глава 4: Карьера и Фриланс](Chapter_4_Career_and_Freelance.md)
   * Повышение эффективности и зарплаты
   * Оптимизация профиля (LinkedIn, GitHub)
   * Фриланс и запуск стартапов
