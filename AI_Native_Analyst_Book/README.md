# Учебник: AI-Native Системный и Бизнес-аналитик

Добро пожаловать в учебник, предназначенный для системных (SA) и бизнес-аналитиков (BA), которые хотят перейти на следующий уровень и использовать AI-агентов в своей повседневной работе. 

В эпоху AI-Native разработки роль аналитика трансформируется: вместо написания скучных ТЗ в Word, аналитик становится "архитектором контекста" и "дирижером", направляющим AI-разработчиков (таких как Claude, Cursor или GitHub Copilot).

   * Настройка рабочего окружения (LLMs, Obsidian, Notion AI)
   * Методология работы: Архитектура контекста
   * Быстрый сбор требований и реверс-инжиниринг с помощью AI

3. [Глава 3: Продвинутая автоматизация аналитики](Chapter_3_Advanced_Automation.md)

4. [Глава 4: Карьера и развитие AI-аналитика](Chapter_4_Career.md)
